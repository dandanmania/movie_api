Documentation Page

This API will provide users with access to information about a few anime movies.
Users will be able to sign up, update their personal information, and create a list of their favorite movies. Apparently.

Action URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Return a list of all movies to the user /movies GET None JSON object with all movies
Return data about a single movie by title /movies/:title GET None JSON object with data about a single movie
Return data about a genre /movies/genre/:genreName GET None JSON with data on the genre
Return data about a director by their name /movies/directors/:direrctorName GET None JSON object with data about the director
Register new user /users POST JSON JSON object about the user, along with an ID
Update User's Name /users/:id PUT None JSON object with updated user data
Add a movie to Favorites list /users/:id/:movieTitle POST None A text message indicating that the movie was added to favorites
Remove a movie from favorites /users/:id/:movieTitle DELETE None A text message indicating that the movie was deleted from favorites
Deregister a user /users/:id DELETE None Text message indicating that the user was removed
