Documentation Page

This API will provide users with access to information about a few anime movies.
Users will be able to sign up, update their personal information, and create a list of their favorite movies. Apparently.

Add a movie to Favorites list /users/:username/movies/:movieTitle POST None A text message indicating that the movie was added to favorites "625f53e803d0dfbe2f8f2f4a has been added to dandanmania's Favorites."
Remove a movie from favorites /users/:username/movies/:movieTitle DELETE None A text message indicating that the movie was deleted from favorites "625f53e803d0dfbe2f8f2f4a has been deleted from dandanmania's Favorites."
Deregister a user /users/:username DELETE None Text message indicating that the user was removed "dandanmania was deleted."
